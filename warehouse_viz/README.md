# Warehouse Visualization System

A 3D interactive warehouse management system built with React, Three.js, and Tailwind CSS.

## Getting Started

1.  Navigate to the project directory:
    ```bash
    cd warehouse_viz
    ```

2.  Install dependencies (if you haven't already):
    ```bash
    npm install
    ```

3.  Start the development server:
    ```bash
    npm run dev
    ```

4.  Open your browser at the URL shown (usually `http://localhost:5173`).

## Features
- **3D View**: Rotate and zoom around the warehouse floor.
- **Add Objects**: Use the sidebar to add Shelves, Fridges, and Freezers.
- **Selection**: Click objects to select them (highlighted blue).
- **Delete**: Remove selected objects via the sidebar.

## Controls
- **Left Click + Drag**: Rotate Camera
- **Right Click + Drag**: Pan Camera
- **Scroll**: Zoom
- **Click Object**: Select
